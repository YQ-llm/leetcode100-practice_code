# 2.字母异位词分组
给你一个字符串数组，请你将 字母异位词 组合在一起。可以按任意顺序返回结果列表。\
哈希表 + 排序
## 算法思路
1.初始化：创建一个哈希表（字典）。键（Key） 存身份证（排序后的字符串），值（Value） 存一个列表（包含所有拥有这张身份证的原单词）。\
2.遍历数组：拿出每一个字符串。\
3.制作身份证：将当前字符串的字母打散并排序，然后重新拼接成一个新的字符串。\
4.存入哈希表：\
&emsp;&emsp;如果这张身份证已经在字典里了，就把原单词追加到对应的列表后面。\
&emsp;&emsp;如果不在，就新建一个列表并放入原单词。\
5.返回结果：遍历结束后，把字典里所有的 Value（也就是分好组的列表集合）打包返回。
## Python3代码实现
```python
import collections
from typing import List

class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        # 使用 defaultdict，当 key 不存在时，会自动初始化为一个空列表 []
        # 格式为 { "排序后的字符串": ["原字符串1", "原字符串2"] }
        anagram_dict = collections.defaultdict(list)
        
        for s in strs:
            # 1. 制作“身份证”：排序字符串
            # sorted(s) 会返回一个按字母排序的列表，如 ['a', 'e', 't']
            # "".join() 会把列表重新拼成字符串 "aet"
            sorted_key = "".join(sorted(s))
            
            # 2. 将原字符串放入对应的篮子里
            anagram_dict[sorted_key].append(s)
            
        # 3. 提取所有的分类结果
        # .values() 会返回 dict_values([["eat","tea"], ["tan"], ...])
        # 用 list() 将其转换为标准的二维列表
        return list(anagram_dict.values())

# --- 测试代码 ---
if __name__ == "__main__":
    sol = Solution()
    
    # 测试用例 1: 常规情况
    strs1 = ["eat", "tea", "tan", "ate", "nat", "bat"]
    print(f"输入: {strs1}")
    print(f"输出: {sol.groupAnagrams(strs1)}")
    # 预期输出: [['eat', 'tea', 'ate'], ['tan', 'nat'], ['bat']] (顺序不重要)
    
    # 测试用例 2: 包含空字符串
    strs2 = [""]
    print(f"\n输入: {strs2}")
    print(f"输出: {sol.groupAnagrams(strs2)}")
    # 预期输出: [[""]]
    
    # 测试用例 3: 只有一个字符
    strs3 = ["a"]
    print(f"\n输入: {strs3}")
    print(f"输出: {sol.groupAnagrams(strs3)}")
    # 预期输出: [["a"]]
```
## 逻辑图解（以 $strs = ["eat", "tea", "bat"]$ 为例）：
1.初始状态：anagram_dict = {}\
2.第 0 步：取出 "eat"\
&emsp;&emsp;制作身份证：排序得到 "aet"。\
&emsp;&emsp;字典里没有 "aet"，新建列表并加入：anagram_dict = {"aet": ["eat"]}。\
3.第 1 步：取出 "tea"\
&emsp;&emsp;制作身份证：排序得到 "aet"。\
&emsp;&emsp;字典里有 "aet"！ 直接追加到列表后面：anagram_dict = {"aet": ["eat", "tea"]}。\
4.第 2 步：取出 "bat"\
&emsp;&emsp;制作身份证：排序得到 "abt"。\
&emsp;&emsp;字典里没有 "abt"，新建列表并加入：anagram_dict = {"aet": ["eat", "tea"], "abt": ["bat"]}。\
5.结束提取：\
&emsp;&emsp;直接拿出 anagram_dict 的 values：[["eat", "tea"], ["bat"]]。
# 复杂度分析
时间复杂度: $O(N \times K \log K)$。 \
&emsp;&emsp; $N$ 是字符串数组 strs 的长度（要遍历 $N$ 个单词）。\
&emsp;&emsp; $K$ 是字符串的最大长度。对每一个字符串进行 sorted() 排序需要花费 $K \log K$ 的时间。哈希表的插是  $O(1)$ 的。\
空间复杂度: $O(N \times K)$。\
&emsp;&emsp;我们需要把所有的字符串及其对应的键存储在哈希表 $anagram_dict$ 中。在最坏的情况下，没有异位词，我们要存 $N$ 个键和 $N$ 个单词，总共占用 $O(N \times K)$ 的内存空间。
